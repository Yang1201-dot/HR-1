🔧 How the Refresh Fix Works:

  1. Each module page detects if it's loaded standalone (not in iframe)
  2. If standalone, it shows a loading spinner and redirects to dashboard
  3. Dashboard receives the module parameter and loads the module in iframe
  4. URL gets updated to show the correct module page
  5. Theme and navigation are preserved

  • Refresh: Refresh any module page → Should stay in dashboard context with module loaded
  • Bookmark: Bookmark module page → Should open correctly in dashboard context
  • Back/Forward: Use browser buttons → Should work correctly
